  [33X[0;0YThe SageMath Package surface_dynamics from Vincent Delecroix [VDCF] provides

  [6XReturns:[106X  [33X[0;10YA list of lists of Ints.[133X

  [3Xorigami[103X  and  returns  its  result  as  [5XGAP[105X object. It returns (a,t,u) where

  of  [22XZ^2[122X  has  a standard basis consisting of a horizontal vector (a,0) and a
  nonhorizontal  vector (t,u), where a, t, u are integers satisfying 0 < t < a
  and 0 < t.[133X
  
  [4X[32X  Example  [32X[104X
