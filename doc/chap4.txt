  [1X4.1 [33X[0;0YSetting up the database[133X[101X
  
  [33X[0;0YTo  use  the  origami  database, you need an instance of ArangoDB running on
  port  8529.  After  installing  ArangoDB,  you  need to set up a database as
  follows:[133X
  
  [30X    [33X[0;6Yuser: 'origami'[133X
  
  [30X    [33X[0;6Ypassword: none[133X
  
  [30X    [33X[0;6Ydatabase: 'origami'[133X
  
  [33X[0;0YIn  this  database, create three collections called [13Xorigami_representatives[113X,
  [13Xorigamis[113X    and    [13Xveechgroups[113X.    Please    download    the    data    from
  [7Xhttps://www.math.uni-sb.de/ag/weitze/CMS/index.php/de/forschung[107X  and  import
  the  [13Xjson[113X-files  into the corresponding collections. All of this can be done
  in   the   web   interface   of   ArangoDB,   which   you  can  reach  under
  [7Xhttp://127.0.0.1:8529[107X.[133X
